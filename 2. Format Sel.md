# Format Sel di Microsoft Excel

## 1. Mengakses Format Sel
Kamu bisa membuka menu Format Sel dengan beberapa cara:
- **Klik kanan** pada sel â†’ Pilih **Format Cells**
- **Gunakan shortcut**: `Ctrl + 1`
- **Gunakan Ribbon**: Tab **Home** â†’ Grup **Number, Alignment, Font, dll.**

---

## 2. Jenis Format Sel

### a. Format Angka (Number Format)
Digunakan untuk mengatur bagaimana angka ditampilkan:
- **General** â†’ Format default tanpa pemformatan khusus.
- **Number** â†’ Menampilkan angka dengan pemisah ribuan.
- **Currency** â†’ Menampilkan angka dengan simbol mata uang.
- **Accounting** â†’ Mirip Currency, tetapi lebih rapi dalam penyusunan angka.
- **Date & Time** â†’ Menampilkan tanggal & waktu dalam berbagai format.
- **Percentage (%)** â†’ Mengubah angka ke persen (misal, `0.5` menjadi `50%`).
- **Fraction** â†’ Menampilkan angka sebagai pecahan (misal, `0.5` menjadi `1/2`).
- **Scientific** â†’ Menampilkan angka dalam format eksponensial (misal, `123456` menjadi `1.23E+05`).

---

### b. Format Teks (Text Format)
- **Text** â†’ Memaksa semua isi sel dibaca sebagai teks, termasuk angka.
- **Wrap Text** â†’ Membungkus teks agar sesuai dalam satu sel.
- **Merge & Center** â†’ Menggabungkan beberapa sel dan menengahkan teks.

---

### c. Format Tampilan Sel (Cell Appearance)
- **Font** â†’ Mengubah jenis, ukuran, dan warna teks.
- **Alignment** â†’ Mengatur perataan teks (kiri, tengah, kanan).
- **Borders** â†’ Menambahkan garis tepi pada sel.
- **Fill Color** â†’ Mengubah warna latar belakang sel.
- **Conditional Formatting** â†’ Mewarnai sel secara otomatis berdasarkan aturan tertentu.

---

## 3. Shortcut Format Sel
Berikut beberapa shortcut berguna:
- `Ctrl + Shift + $` â†’ Format Currency
- `Ctrl + Shift + %` â†’ Format Percentage
- `Ctrl + Shift + #` â†’ Format Date
- `Ctrl + Shift + @` â†’ Format Time
- `Ctrl + Shift + ^` â†’ Format Scientific
- `Ctrl + Shift + ~` â†’ Format General
